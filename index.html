About a Scrum Master Living in a Developer's World

Hi! I'm Amy Martinez!

Ever since I started my career as a scrum master I've been asking a controversial question, "Does a Scrum Master need to be technical?" I went on an incredible journey to learn C#, HTML and CSS software development in the Eleven Fifty Academy Software Development bootcamp to try to answer this question. I conlcuded that although it's not required, it can be helpful. Take a peek below to see what my colleagues are saying now that I have completed the program. If you are a scrum master working in a software development environment, I encourage you to go on this, or a similar, journey, so that you can continue to grow in your serverant leadership, and remember....

Be Agile!
